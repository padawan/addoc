This page lists the prices (_excluding taxes_) of Private Cloud Configurations. For **more information** and/or a **custom configuration** contact our [Sales Team](https://www.alwaysdata.com/fr/hebergement-infogere/).

For all purchases of a Private Cloud Configuration, the server installation is done after contract validation and payment of the first invoice. There is no installation fee.

{{% notice note %}}
_Private Cloud_ is an offer of managed and managed servers. Our team is responsible for the installation, configuration, and deployment of platforms. You did not manage system administration.
{{%/notice %}}
