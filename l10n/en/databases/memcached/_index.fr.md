## Login

|            |                                                                        |
| ---------- | ---------------------------------------------------------------------- |
| **Server** | localhost or 127.0.0.1 |
| **Port**   | 11211 (default Memcached port)                      |

Memcached only runs locally with isolated privileges (separate user).
